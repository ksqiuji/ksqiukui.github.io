基础 - 游戏模式 (gamemode)

游戏模式概述

gamemode指令用于更改玩家的游戏模式。Minecraft基岩版有四种主要的游戏模式：生存模式、创造模式、冒险模式和旁观模式。

基本语法

gamemode指令的基本格式：/gamemode <模式> [玩家]

基本用法

1. 更改自己的游戏模式：/gamemode creative

2. 更改其他玩家的游戏模式：/gamemode survival 玩家名

3. 使用数字代码更改模式：/gamemode 1 @a

重要提示

在基岩版中，您需要有适当的权限才能使用gamemode指令。在多人游戏中，通常需要管理员或OP权限。

模式代码与数字

在Minecraft基岩版中，可以使用模式名称或数字代码：

• survival 或 0 - 生存模式

• creative 或 1 - 创造模式

• adventure 或 2 - 冒险模式

• spectator 或 3 - 旁观模式 (基岩版1.19.10及以上)

示例

/gamemode 1 - 将自己的模式改为创造模式

/gamemode adventure Steve - 将Steve改为冒险模式

/gamemode 0 @a - 将所有玩家改为生存模式

使用选择器

Minecraft基岩版提供了多种选择器来精确定位玩家：

• @p - 最近的玩家

• @a - 所有玩家

• @r - 随机玩家

• @e - 所有实体 (但gamemode只能用于玩家)

选择器示例

/gamemode creative @a - 将所有玩家改为创造模式

/gamemode survival @r - 将随机玩家改为生存模式

/gamemode adventure @p - 将最近的玩家改为冒险模式

注意事项

1. 在基岩版中，旁观模式需要版本1.19.10或更高

2. 冒险模式可以防止玩家破坏方块，适合制作冒险地图

3. 使用选择器时要注意权限，避免意外更改其他玩家的模式

4. 在命令方块中使用时，确保有适当的红石信号触发

版本兼容性

基岩版和Java版的gamemode指令语法基本相同，但有些细微差别。本文档专注于基岩版用法。
